# Contexto del proyecto

- Análisis de jugadores de fútbol: rendimiento, valor de mercado y factores de riesgo: Este proyecto analiza un conjunto de datos de jugadores profesionales con el objetivo de entender qué variables explican mejor su valor de mercado, su rendimiento y su riesgo de transferencia.

- Se estudian métricas deportivas como goles, asistencias y minutos jugados, junto con variables de perfil como edad, posición, nacionalidad, estado de contrato y propensión a lesiones.

- El objetivo es simular un caso de uso realista de scouting y valoración de jugadores, similar al que podrían utilizar clubes o analistas deportivos.

## Preguntas a resolver

El análisis se centra en responder las siguientes preguntas:

¿Qué factores explican el valor de mercado de un jugador?
¿Existe una relación clara entre rendimiento deportivo y valor económico?
¿Qué posiciones tienen mayor valor en el mercado?
¿Cómo influye la edad en la valoración de los jugadores?
¿Qué relación existe entre riesgo de transferencia y valor de mercado?
¿Cómo se distribuye el talento entre clubes y nacionalidades?

## A partir del análisis se pueden extraer las siguientes conclusiones

El valor de mercado no depende de una sola variable, sino de una combinación de rendimiento, edad, posición y contexto contractual.
Existe una clara segmentación del mercado entre jugadores élite y el resto.
La edad juega un papel crítico, definiendo una ventana óptima de valorización.
Las posiciones ofensivas tienden a generar mayor dispersión de valor.
El mercado de jugadores es altamente desigual tanto por clubes como por nacionalidad.

Este proyecto simula un caso real de análisis de scouting deportivo, integrando:

Análisis exploratorio de datos (EDA)
Agregaciones por grupos (clubes, nacionalidad, posición)
Visualización avanzada de datos
Interpretación orientada a negocio deportivo
